# Linux Server Automation Toolkit

## Overview
Linux Server Automation Toolkit is a Bash scripting project that automates common Linux system administration tasks. It is designed to improve efficiency by reducing manual work and demonstrates practical Linux and DevOps skills.

## Features
- Create Linux users automatically
- Backup directories with timestamped archives
- Monitor CPU, memory, and system uptime
- Check disk usage and warn when usage is high
- Archive log files
- Generate a complete system health report
- Menu-driven interface for easy operation

## Technologies Used
- Linux (Ubuntu)
- Bash Shell Scripting
- Git
- Linux Commands (tar, df, free, uptime, ps, who, useradd)
